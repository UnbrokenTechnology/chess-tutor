# Android app (deprioritised — Phase 5)

Kotlin + Jetpack Compose, JNI bindings generated by `uniffi`. No Android hardware on the team at the moment, so this ships after Apple and Windows are solid.

## Expected layout

```
android/
├── app/
│   └── src/main/
│       ├── kotlin/       Compose UI, view models
│       ├── jniLibs/      .so files per ABI (copied by scripts/build-android-libs.sh)
│       └── assets/       opening book, ECO mapping
└── core/                 generated uniffi Kotlin bindings
```

## Supported ABIs

- `arm64-v8a`   → `aarch64-linux-android`
- `armeabi-v7a` → `armv7-linux-androideabi`
- `x86_64`      → `x86_64-linux-android` (emulator)
